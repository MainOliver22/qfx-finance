# **App Name**: QFX Finance

## Core Features:

- Secure User Authentication: Robust user registration and login with OAuth 2.0 and username/password, leveraging session-based authentication for a secure banking experience.
- Fiat Account Dashboard: Mimic Chase Bank's online portal with a clean dashboard for viewing mock fiat account balances, recent transactions, and spending insights.
- Web3 Wallet Management: Seamless integration to connect and manage multiple Web3 wallets, including MetaMask, WalletConnect, and Coinbase Wallet, to display crypto assets.
- Real-time Crypto Portfolio & Trading: Display user crypto holdings with real-time price charts (leveraging Recharts/TradingView data) and fundamental buy/sell functionality for key cryptocurrencies.
- AI Financial Insight Tool: Provide personalized AI-generated market summaries and investment suggestions, acting as a tool to aid in financial decision-making.
- Investment Package Enrollment: A streamlined interface for users to browse predefined investment packages and easily enroll, tracking their participation.

## Style Guidelines:

- Background Color: Deep Navy Blue (#0A2540) serves as the primary dark backdrop, evoking institutional trust and a sophisticated digital ambiance. Its deep saturation provides an immersive feel.
- Primary Color: Chase Blue (#117ACA) is strategically used for interactive elements, primary call-to-actions, and brand-identifying features. Its vibrancy ensures clarity and engagement against the dark background.
- Accent Color: Electric Blue (#00D4AA) is dedicated to all cryptocurrency and Web3-related modules. This analogous hue provides a fresh, modern 'crypto-native' accent, signaling innovation.
- Neutral Color: Pure White (#FFFFFF) for text, critical information, and distinct UI components, ensuring maximum readability and crispness on the dark canvas.
- Headlines and Body Text: 'Inter' (sans-serif) offers a clean, objective, and modern aesthetic, balancing legibility for long-form content with a sharp look for titles.
- Monospace Font: 'Source Code Pro' (monospace) is designated for displaying wallet addresses, transaction hashes, and other code-like identifiers, ensuring clarity and distinction.
- Use a set of sharp, minimalist icons with a modern financial and blockchain-aligned style, ensuring consistency and immediate recognition for actions and information.
- Adopt a 'Chase Bank-style' layout featuring persistent sidebar navigation, modular dashboard cards for key data, and clean, tabular transaction displays, all within precise grid structures for organizational clarity.
- Subtle Glassmorphism: Integrate delicate frosted glass effects into crypto-specific modules, subtly differentiating them from traditional banking UI elements while adding a modern, Web3-inspired visual texture.
- Incorporate subtle, performant transitions and micro-interactions for state changes, navigation, and data updates, providing a fluid and premium user experience without distraction.
